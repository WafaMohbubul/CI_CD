First iteration:

  1. Launch EC2 Instance
  2. Add relevant security group to Ec2 instance
  3. Create a third job in jenkins to SSH into environment
  4. Copy code into environment
  5. Install nginx
  6. Share public IP and github document with screenshots of third job and execute shells commands
  7. MUST HAVE A DIAGRAM for CI CD in your repo.

Trigger the third job manually, so you can test if it works. Once you know it works, add it to post build actions of the second job.

Second iteration:

Launch the app - should be triggered if third job was successful

Third iteration:

Create 5th job to set up mongodb in a new EC2 instance if fourth job is successful.
